# tasks.md

> UniMeet — 개발 작업 계획 문서  
> 문서 버전: v1.0 | 기준: 7~8주차 MVP 완성, 발표 시연

---

## 1. 개발 목표 요약

| 항목 | 내용 |
|---|---|
| **최종 목표** | 유학생이 한국어 없이 동아리를 탐색·지원하고, 운영진이 지원자·회비를 관리할 수 있는 MVP 앱 완성 |
| **개발 기간** | 7주차 ~ 8주차 (총 2주) |
| **팀 구성** | 3인 (역할 분담은 섹션 5 참조) |
| **시연 환경** | Expo Go (QR 코드 즉시 실행) |
| **핵심 원칙** | 기능 욕심보다 완성도 우선 — Must Have 5개 기능을 완벽히 시연하는 것이 목표 |

## 9. 최종 점검 체크리스트

### 기능 동작 확인

- [ ] @sunmoon.ac.kr 이메일로만 회원가입 가능한가?
- [ ] 이메일 인증 후 로그인이 정상 동작하는가?
- [ ] 선호 언어 설정이 저장되고 동아리 목록에 반영되는가?
- [ ] 동아리 소개가 선호 언어로 번역되어 표시되는가?
- [ ] 번역 결과가 캐시되어 재진입 시 API 재호출이 없는가?
- [ ] 동아리 필터(분야·유학생 환영)가 정상 동작하는가?
- [ ] 지원서를 작성하고 제출할 수 있는가?
- [ ] 동일 동아리에 중복 지원이 차단되는가?
- [ ] 운영진 계정에서 지원자 목록을 확인할 수 있는가?
- [ ] 합격 처리 시 지원자에게 FCM 푸시 알림이 발송되는가?
- [ ] 회비 입출금 내역을 등록하고 조회할 수 있는가?
- [ ] 미납 부원에게 알림을 발송할 수 있는가?

### 시연 환경 확인

- [ ] Expo Go 앱이 최신 버전으로 설치되어 있는가?
- [ ] QR 코드로 앱이 정상 실행되는가?
- [ ] 시연용 유학생 계정이 준비되어 있는가?
- [ ] 시연용 운영진 계정이 준비되어 있는가?
- [ ] Firestore에 샘플 동아리 5건 이상 입력되어 있는가?
- [ ] 인터넷 연결 환경이 발표 장소에서 확인되었는가?
- [ ] iOS · Android 양 플랫폼에서 테스트가 완료되었는가?

### 발표 자료 확인

- [ ] 슬라이드 10장이 완성되어 있는가?
- [ ] 시연 스크립트가 시나리오별로 작성되어 있는가?
- [ ] 팀원 모두 시나리오를 숙지하고 있는가?
- [ ] 리허설을 2회 이상 완료했는가?
- [ ] 발표 자료 파일이 PDF·PPTX 두 형식으로 백업되어 있는가?
