# UniMeet (유니밋)

> 선문대학교 내·외국인 학생을 위한 동아리 탐색 및 지원 플랫폼

UniMeet은 선문대학교 학생들이 동아리를 쉽게 찾고, 다국어로 동아리 정보를 확인하며, 앱 안에서 지원서를 제출할 수 있도록 돕는 모바일 앱입니다. 동아리 회장과 운영자는 홍보카드, 지원서, 권한 관리 흐름을 한 곳에서 관리할 수 있습니다.

## 발표용 MVP 상태

현재 구현은 **mock 데이터를 사용하는 발표용 MVP 앱 화면**입니다. Firebase 연동 전 단계이지만, 실제 서비스 흐름을 설명하고 시연할 수 있도록 주요 화면과 인터랙션을 연결했습니다.

현재 가능한 시연 흐름:

- 홈에서 서비스 핵심 현황 확인
- 동아리 탐색 화면에서 동아리 카드 확인
- 동아리 상세 화면에서 대표 사진, 소개, 모집 요강, 최근 활동 확인
- 지원서 작성 화면 진입
- 지원내역에서 지원 상태 확인
- 알림 목록, 읽음 처리, 알림 상세, 알림 삭제 확인
- 마이페이지에서 회장/관리자 대시보드 진입
- 언어 설정에서 UI 언어와 콘텐츠 번역 설정 변경

## 주요 기능

### 1. 홈

홈은 학생이 앱을 열었을 때 가장 먼저 보는 요약 화면입니다.

- 사용자 환영 메시지
- 동아리 탐색 CTA
- 회장 대시보드 진입 버튼
- 내 동아리 수, 지원서 수, 안 읽은 알림 수 표시
- 각 요약 카드 클릭 시 관련 화면으로 이동
- 최근 지원 상태 요약
- 추천 동아리 카드
- 공지 피드 mock

### 2. 동아리 탐색

동아리 탐색 화면에서는 홍보카드 형태로 동아리를 확인할 수 있습니다.

- 동아리 검색
- 모집 중, 카테고리, 언어 필터
- 동아리 대표 이미지, 소개, 모집 상태, 검토 상태 배지
- 동아리 카드 클릭 시 상세 화면 이동

### 3. 동아리 상세

동아리 상세 화면은 발표 시 핵심 시연 화면입니다.

- 대표 사진
- 동아리명 및 영문명
- 멤버 수, 모집 상태, 지원 언어 요약
- 운영자 검토 상태 배지
- 동아리 소개
- 모집 요강
- 정기 모임 정보
- 최근 활동 목록
- 지원하기 버튼
- 원문과 번역문 동시 표시

### 4. 지원서 작성 및 지원내역

지원서 기능은 현재 mock 플로우로 구성되어 있습니다.

- 이름과 학번 기본 정보 표시
- 회장이 설정한 질문 표시
- 단답형, 장문형, 객관식, 체크박스, 파일 첨부 타입을 고려한 구조
- 제출 시점의 양식 스냅샷을 저장한다는 UX 안내
- 지원내역 화면에서 지원 상태 확인

### 5. 알림

알림은 Zustand store 기반으로 동작합니다.

- 안 읽은 알림 목록
- 전체 알림 목록
- 모두 읽음 처리
- 알림 클릭 시 상세 화면 이동
- 알림 클릭 시 읽음 처리
- 알림 상세에서 관련 화면 이동
- 알림 삭제

### 6. 언어 설정과 번역 정책

UniMeet은 **앱 UI 언어**와 **콘텐츠 번역 언어**를 분리합니다.

- UI 언어: 버튼, 탭, 메뉴, 라벨에 적용
- 콘텐츠 번역 언어: 동아리 소개, 모집 요강, 공지에 적용
- 콘텐츠 번역 사용/미사용 토글 제공
- 번역 OFF 시 원문만 표시
- 번역 ON 시 원문 + 선택 언어 번역문 표시

현재 mock 지원 언어:

- 한국어
- English
- 日本語
- 中文
- Tiếng Việt

실서비스 번역 흐름은 아래 구조를 권장합니다.

```text
App
  -> Firebase Callable Function
  -> Google Cloud Translation Advanced
  -> Firestore 번역 캐시
  -> App에서 캐시된 번역 표시
```

Google Translation API 키는 앱 클라이언트에 직접 노출하지 않고, Firebase Functions에서 호출하는 방식으로 구현합니다.

### 7. 회장 대시보드

회장 대시보드는 동아리 운영자가 지원자와 홍보 정보를 관리하는 화면입니다.

- 신규 지원자 수
- 지원서 질문 수
- 홍보카드 검토 상태
- 홍보카드 수정 진입
- 지원서 양식 설정 진입
- 지원자 목록
- 합격/불합격 처리 버튼 mock

### 8. 관리자 대시보드

관리자 대시보드는 운영자/학교 관리자 기능을 앱 안에서 확인하는 화면입니다.

- 동아리 수
- 승인 대기 수
- 감사 로그 수
- 동아리 생성
- 회장 권한 부여
- 운영자 승인
- 감사 로그 목록

## 기술 스택

| 영역 | 기술 |
|---|---|
| 앱 | React Native + Expo |
| 라우팅 | Expo Router |
| 상태 관리 | Zustand |
| UI | React Native StyleSheet + Expo Vector Icons |
| 인증 예정 | Firebase Authentication |
| 데이터베이스 예정 | Cloud Firestore |
| 파일 저장 예정 | Firebase Storage |
| 번역 예정 | Firebase Callable Functions + Google Cloud Translation Advanced |

## 프로젝트 구조

```text
app/
  (tabs)/              학생용 하단 탭 화면
  admin/               관리자 대시보드 mock
  club/[id]/           동아리 상세 및 지원서 작성 화면
  leader/              회장 대시보드 mock
  notifications/[id]   알림 상세 화면
  settings/language    언어 설정 화면
  my-clubs             내 동아리 화면
components/            공통 UI 컴포넌트
constants/             디자인 토큰
data/                  mock 데이터
store/                 Zustand 상태 관리
types/                 공통 TypeScript 타입
utils/                 유틸리티 및 i18n 헬퍼
```

## MVP 범위

MVP 방향에 포함된 기능:

- 학교 이메일 기반 회원가입/로그인
- 언어 선택
- 동아리 탐색
- 동아리 상세 페이지
- 지원서 제출
- 지원 상태 확인
- 회장의 홍보카드 및 지원자 관리
- 운영자의 동아리 생성 및 회장 권한 부여
- 앱 내부 알림

이후 단계로 미룬 기능:

- 실시간 채팅
- 푸시 알림
- 행정 서류 제출 및 검토
- 통계 대시보드
- 신고/차단 워크플로우
- 관리자 웹 대시보드
- 실제 파일 업로드 플로우
- Firebase Auth, Firestore, Storage 연동
- Google Translation API 실연동
