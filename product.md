# product.md

> 문서 버전: v1.0 | 작성 기준: 사전기획서 최종본

---

## 1. 서비스 개요

**UniMeet**은 선문대학교 재학생(내국인 + 유학생)이 동아리를 탐색하고 지원하며, 운영진이 부원을 관리할 수 있는 캠퍼스 통합 모바일 앱이다. 동아리 공고를 다국어로 자동 번역하여 88개국 유학생이 한국어 없이도 교내 동아리에 참여할 수 있도록 진입 장벽을 낮춘다. React Native / Expo 기반으로 개발하며, 이번 학기 7~8주차 2주 내 MVP를 완성하는 것을 목표로 한다.

---

## 2. 문제 정의

선문대학교는 88개국 출신 유학생이 재학 중이나, 교내 동아리 모집 공고는 전부 한국어로만 제공되어 유학생이 내용을 파악하거나 지원하기 어렵다. 내국인 학생은 글로벌 교류를 원하지만 유학생과 연결되는 교내 채널이 없다. 동아리 운영진은 회비·출석·공지를 카카오톡·구글폼·계좌이체로 분산 관리하여 행정 부담이 크다.

> **문제 한 줄 요약:** 동아리가 내국인 중심으로 설계되어 유학생은 참여하기 어렵고, 운영진도 분산된 도구로 인해 비효율적으로 운영된다.

---

## 3. 타깃 사용자

| 구분 | 대상 | 핵심 니즈 |
|---|---|---|
| **1차 타깃** | 선문대학교 재학 유학생 (88개국) | 모국어로 동아리 정보 탐색, 간편한 지원 절차 |
| **2차 타깃** | 글로벌 교류에 관심 있는 내국인 재학생 | 유학생과 연결되는 교내 채널 |
| **3차 타깃** | 동아리 운영진 | 회비·출석·공지 통합 관리, 홍보 채널 확보 |

**페르소나 예시 1 — 유학생:** 이란 출신 재학생, 한국어 중상급. 한국 친구를 사귀고 싶지만 동아리 공고를 이해하기 어렵고 지원 절차가 불투명하다.

**페르소나 예시 2 — 동아리장:** 4학년, 매 학기 회비 내역을 엑셀로 수작업 관리. 미납자 추적, 공지 발송, 홍보 모두 개별 앱을 사용해 행정 부담이 크다.

## 4. 핵심 가치 제안 (Value Proposition)

| 가치 | 설명 |
|---|---|
| **언어 장벽 제거** | 동아리 공고 다국어 자동 번역 (Google Translate API)으로 유학생이 모국어로 탐색·지원 가능 |
| **신뢰 기반 커뮤니티** | 대학 이메일(@sunmoon.ac.kr) 실명 인증으로 교내 구성원만 참여 가능한 안전한 환경 제공 |
| **동아리 운영 자동화** | 회비·출석·공지를 앱 하나로 통합하여 운영진의 행정 부담 절감 |

## 5. 사용자 시나리오

### 시나리오 A — 유학생의 동아리 탐색 및 지원
1. 이란인 유학생이 UniMeet에 대학 이메일로 가입
2. 언어 필터(이란어)를 설정하고 동아리 목록 탐색
3. '유학생 환영' 배지가 있는 동아리의 모집 공고를 모국어로 열람
4. 앱 내 지원서를 작성하고 제출 → 운영진으로부터 합격 알림 수신

### 시나리오 B — 내국인 학생의 글로벌 교류 탐색
1. 내국인 3학년이 '유학생 환영' 필터로 동아리 검색
2. 다국적 부원이 있는 문화 교류 동아리 발견 후 지원
3. 동아리 가입 후 유학생과 관심사 기반으로 연결

### 시나리오 C — 운영진의 부원 관리
1. 동아리장이 앱에서 신규 지원자 목록 확인 후 합격 처리
2. 회비 납부 현황 대시보드에서 미납자에게 자동 알림 발송
3. 공지사항 작성 후 전체 부원에게 다국어 번역 푸시 알림 전송

## 7. 경쟁 서비스 대비 차별점

| 서비스 | 한계 | UniMeet 차별점 |
|---|---|---|
| **에브리타임** | 익명 구조, 다국어 지원 없음, 동아리 관리 툴 없음 | 실명 인증 + 다국어 번역 + 동아리 관리 내장 |
| **헬로우톡** | 일반 언어 교환 앱, 교내 커뮤니티 기능 없음 | 교내 구성원 한정, 동아리 중심 연결 |
| **카카오톡** | 범용 메신저, 동아리 특화 기능 없음 | 동아리 지원·관리·공지 통합 특화 기능 |

## 8. 비즈니스 모델 개요

수익화는 3단계로 구성한다. 초기에는 유저 확보와 플랫폼 신뢰도 확립에 집중하고, 이후 B2B 확장과 광고로 수익을 다각화한다.

| 단계 | 수익 방식 | 지불 주체 |
|---|---|---|
| **Phase 1** | 오픈 이벤트 참가비 5% 수수료 | 이벤트 참가 학생 |
| **Phase 2** | 총동아리회·대학 본부 B2B SaaS 구독 (월 10~30만 원) | 총동아리회 또는 대학 본부 |
| **Phase 3** | 대학생 타겟 버티컬 광고 (어학원, 유학 브랜드 등) | 광고주 |

## 9. MVP 범위

**이번 학기 핵심 가설:** 유학생이 한국어 없이도 동아리를 탐색하고 지원할 수 있는가?

| MVP 기능 | 구현 주차 |
|---|---|
| 이메일 인증 & 회원가입 | 7주차 |
| 동아리 목록 탐색 (다국어 번역 포함) | 7주차 |
| 동아리 상세 페이지 | 8주차 |
| 가입 신청서 제출 & 합격 알림 | 8주차 |
| 회비 내역 관리 페이지 | 8주차 |

**발표 시연 시나리오:**  
유학생 계정으로 로그인 → 언어 필터로 동아리 탐색 → 상세 페이지 열람 → 지원서 제출 → 운영진 계정에서 합격 처리

## 10. 향후 확장 방향

- **v1.1:** 토스페이먼츠 PG 연동으로 이벤트 참가비 인앱 결제 지원
- **v1.2:** GPT API 기반 AI 아이스브레이킹 기능 추가
- **v2.0:** 총동아리회 전용 관리자 페이지 및 매너 온도·상호 평가 시스템
- **다대학 확장:** 선문대 검증 후 외국인 유학생 비율 높은 타 대학으로 라이선스 확장
- **광고 플랫폼화:** 유학생·내국인 구분 타겟팅 광고 (에브리타임 대비 정교한 타겟팅)
